
 ◆ SVG 이미지
 웹에서 사용할 수 있는 이미지 파일 형식은 gif, jpg/jpeg, png 가 있다고 했습니다. 그런데 최근에 브라우저에서 svg 파일을
 지원하게 되면서 아이콘이나 로고 이미지에 svg 이미지 파일 형식이 많이 사용되고 있습니다.

 1) SVG 파일 형식이란?
 요즘 웹 사이트를 표시할 화면 크기가 다양하기 때문에 같은 이미지라도 작게 표현해야할 때가 있고 크게 표현해야할 때가 있습니다.
 그런데 원래 작은 크기였던 gif 나 jpg/jpeg, png 파일을 크게 확대해 나타내면 이미지 테두리 부분이 울퉁불퉁해집니다. 이런
 이미지를 비트맵 이미지라고 합니다. 반면, 이미지를 아무리 확대하거나 축소해도 원래의 깨끗한 상태 그대로 유지되는 이미지를
 벡터 이미지(vector image)라고 하는데 그런 이미지가 SVG 이미지입니다.

 SVG 이미지는 크기를 조정해도 이미지가 깨지지 않고 깨끗이 유지되기 때문에 로고나 아이콘에서 많이 사용됩니다.
 SVG 이미지는 <img> 태그를 이용해 파일 형태로 삽입할 수도 있지만 태그를 이용해 직접 만들수도 있습니다.
 즉 SVG는 이미지이면서도 소스 코딩을 통해 만들고 편집할 수 있기 때문에 HTML5 등장과 함께 많은 주목을 받고 있습니다.

 2) SVG 이미지 삽입하기
 SVG 이미지는 직접 만들수도 있지만 미리 만들어 놓은 SVG 이미지를 삽입할 수도 있습니다. SVG 이미지는 어도비 일러스트레이터 등
 디자인 프로그램을 통해 만들 수 있습니다.

 3) SVG 파일을 지원하지 않는 브라우저를 위해
 대부분의 모던 브라우저에서는 svg 파일을 지원하지만 인터넷 익스플로러 8 이하를 고려해야 하거나 안드로이드 2.3 이하 버전에서는
 svg 파일을 표시할 수 없습니다. 이런 경우, svg 이미지 파일 대신 png 이미지 파일을 사용해야 합니다.

 브라우저에서 특정 기능을 지원하는지 여부를 테스트해주는 툴이 있는데 Modernizer 입니다
