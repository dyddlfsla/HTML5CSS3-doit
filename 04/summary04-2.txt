
 ◆ 사용자 입력을 위한 <input> 태그
 아이디나 검색어를 입력하는 텍스트 상자나 로그인 버튼처럼 사용자가 입력하는 부분은 사실 대부분 <input> 태그를 이용해 넣습니다.
 <input> 태그로 넣을 수 있는 항목은 아이디나 비밀번호를 비롯해 이메일 주소, 전화번호, 날짜, 시간, 이미지, 버튼 등 다양합니다.
 이렇게 다양한 옵션 중에서 무엇이 될지는 type 속성에 의해 결정이 됩니다.

 1) <input> 태그 살펴보기
 우선 <input> 태그는 폼에서 사용자 입력을 받기 위해 사용하는 태그라는 것을 기억해야 합니다.

 2) <input> 태그 - 입력 항목 만들기
 웹에서의 폼은 크게 사용자가 입력하는 부분과 입력한 내용을 서버로 보내는 버튼 부분으로 나눌 수 있습니다.
 <input> 태그로 만들 수 있는 폼 요소가 상당히 많은데 그것은 <input> 태그 안에 있는 type 속성을 이용해 구분합니다.
 예를 들어, <input type="text">로 지정하면 한 줄짜리 텍스트를 입력하는 필드가 만들어지고 <input type="checkbox">
 라고 하면 체크박스가 삽입되는 식입니다.

  ① id 속성 사용하기
  폼을 만들다 보면 똑같은 폼 요소가 여러 번 사용됩니다. 이렇게 여러 번 사용된 폼 요소를 구분하기 위해 사용하는 것이
  id 속성입니다. id 속성값은 최소한 한 개 이상의 문자여야 하고 공백이 있어서는 안됩니다.

  ② <input> 태그의 type 속성에서 사용 가능한 유형
  앞에서 나왔던 로그인 창에서처럼 아이디나 비밀번호를 입력하거나 로그인 버튼을 클릭하는 등 <input> 태그를 사용해
  사용자가 입력할 수 있는 형태는 매우 다양합니다. 내용이 조금 많아 보일 수 있으나 여기 나열된 유형은 자주 사용되는 것이므로
  모두 알아두는 것이 좋습니다.

   - hidden : 사용자에게는 보이지 않지만 서버로 넘겨지는 값을 가집니다.
   - text : 한 줄짜리 텍스트를 입력할 수 있는 텍스트 상자를 넣습니다.
   - search : 검색 상자를 넣습니다.
   - tel : 전화번호 입력 필드를 넣습니다.
   - url : URL 주소를 입력할 수 있는 필드를 넣습니다.
   - email : 메일 주소를 입력할 수 있는 필드를 넣습니다.
   - password : 비밀번호를 입력할 수 있는 필드를 넣습니다.
   - datetime : 국제 표준시(UTC)로 설정된 날짜와 시간(연, 월, 일, 시, 분, 초, 분할 초)를 넣습니다.
   - datetime-local : 사용자가 있는 지역을 기준으로 날짜와 시간을 넣습니다.
   - date : 사용자가 있는 지역을 기준으로 날짜(연, 월, 일)를 넣습니다.
   - month : 사용자가 있는 지역을 기준으로 날짜(연, 월)를 넣습니다.
   - week : 사용자가 있는 지역을 기준으로 날짜(연, 주)를 넣습니다.
   - time : 사용자가 있는 지역을 기준으로 시간(시, 분, 초, 분할 초)를 넣습니다.
   - number :  숫자를 조절할 수 있는 화살표를 넣습니다.
   - range : 숫자를 조절할 수 있는 슬라이드 막대를 넣습니다.
   - color : 색상 표를 넣습니다.
   - checkbox : 주어진 항목에서 2개 이상 선택 가능한 체크박스를 넣습니다.
   - radio : 주어진 항목에서 1개만 선택 가능한 라디오 버튼을 넣습니다.
   - file : 파일을 첨부할 수 있는 버튼을 넣습니다.
   - submit : 서버 전송 버튼을 넣습니다.
   - image : submit 버튼 대신 사용할 이미지를 넣습니다.
   - reset : 리셋 버튼을 넣습니다.
   - button : 아무 기능이 없는 순수 버튼을 넣습니다.

 3) type="hidden" - 히든 필드 만들기
 먼저 히든(hidden) 필드는 화면상의 폼에는 보이지 않지만 사용자가 입력을 마치고 폼을 서버로 전송할 때 폼 데이터와 함께
 전송되는 요소입니다. 보통 사용자에게 굳이 보여줄 필요가 없지만 관리자가 알아야 하는 것을 히든 필드로 입력합니다.

 4) type="text" - 텍스트 필드 만들기
 텍스트 필드는 한 줄짜리 일반 텍스트를 입력하는 필드입니다. 폼에서 가장 많이 사용하는 요소로 주로 아이디나, 주소, 이름 등
 의 텍스트를 입력합니다. 텍스트 필드에서 사용할 수 있는 속성에는 다음과 같은 것들이 있습니다.

  ① name : 텍스트 필드를 구별할 수 있도록 이름을 붙입니다.
  ② size : 텍스트 필드의 길이를 지정합니다. 즉 화면에 몇 글자가 보이도록 할 것인지를 지정합니다. 예를 들어 최대 입력가능한
           글자 수가 10자인데 size 를 5로 지정하면 텍스트 필드 크기가 5개 글자 크기에 맞추어져 나머지 5개 글자는 화면에 보이지 않습니다.
  ③ value : 텍스트 필드 요소가 화면에 표시될 때 텍스트 필드 부분에 표시될 내용입니다. 이 속성을 사용하지 않으면 빈 텍스트 필드가
            표시됩니다.
  ④ maxlength : 텍스트 필드에 입력할 수 있는 최대 문자 개수를 지정합니다.

 4) type="password" - 비밀번호 입력란 만들기
 "password" 는 말그대로 비밀번호 입력란을 만듭니다. 하지만 아이디와는 달리 화면에 직접적으로 보이면 안되겠죠 그래서 패스워드 필드는
 텍스트 필드과 거의 같지만 사용자가 입력하는 내용이 화면에 표시되지 않고 '*' 처럼 표시됩니다.

 5) type="search", type="url", type="email" ... - 분화된 텍스트 필드
 이 필드들은 텍스트 필드에서 시작된 필드이므로 텍스트 필드와 동일한 속성을 사용합니다.

 6) type="number" - 숫자 입력하기
 이 속성을 사용하면 사용자가 입력한 내용을 숫자로 인식합니다. 브라우저에 따라 스핀박스가 생길 수 있습니다.

 7) type="range" - 슬라이드 막대로 숫자 지정하기
 이 속성은 슬라이드 막대를 움직여 숫자 값을 입력하게 합니다. 사용할 수 있는 속성은 다음과 같습니다.
  ① min : 필드에 입력할 수 있는 최솟값을 지정합니다. 기본값은 0입니다.
  ② max : 필드에 입력할 수 있는 최댓값을 지정합니다. 기본값은 100입니다.
  ③ step : 짝수나 홀수 등 특정 숫자로 제한하려고 할 때 숫자 간격을 지정할 수 있습니다. 기본값은 1입니다.
  ④ value : 필드에 표시할 초기값입니다.

 8) type="radio" type="checkbox" - 라디오 버튼과 체크박스 넣기
 라디오 버튼과 체크박스는 여러 항목 중 원하는 항목을 선택할 때 사용하는 폼 요소입니다. 이 때 한 개만 선택하도록 할 경우,
 라디오 버튼을 사용하고 두 개 이상의 여러 가지를 선택해도 될 경우, 체크박스를 사용합니다.

  ① name : 라디오 버튼이나 체크박스가 여러 개 있을 경우, 서버에서 구분하기 위해 이름을 지정합니다. 라디오 버튼의 경우
  여러 개 중에서 하나를 지정하는 것이기 때문에 관련 있는 그룹끼리는 모두 같은 name 을 사용해야 합니다.
  ② value : 서버로 넘길 때 실제로 전송되는 값입니다. 이 값은 영문이거나 숫자여야 하며 필수 속성입니다.
  ③ checked : 기본적으로 선택해놓고 싶은 항목이 있다면 사용합니다.

 9) type="color" - 색상 선택 표 상자 넣기
 color 필드는 색상표에서 사용자가 직접 색상을 선택할 수 있게 해줍니다. 색상 값은 16진수로 표시하고 'red' 나 'yellow'
 같은 이름은 사용할 수 없습니다.

 10) type="date", "month", "week" - 날짜 표시하기
 웹 문서나 애플리케이션에 달력을 넣고 싶다면 해당 속성을 이용하면 됩니다.

 11) type="time", "datetime", "datetime-local" - 시간 지정하기

 12) type="submit", "reset", "button" - 서버 전송, 리셋 버튼 넣기
 리셋 버튼은 <input> 요소에 입력된 모든 정보를 재설정해 사용자가 입력한 내용을 모두 지울 수 있습니다. 이때 value 속성을 사용해
 버튼에 표시할 내용을 지정합니다. 또한 submit 버튼으로 전송된 정보는 처음에 <form> 태그의 action 에서 지정한 프로그램으로
 넘겨집니다.

 13) type="image" 이미지 버튼 넣기
 type="image"를 사용하면 submit 버튼 대신에 이미지를 넣을 수 있습니다.

 14) type="button" 순수 버튼 넣기
 폼 안에 버튼 형태를 만듭니다. 이 버튼은 submit 이나 reset 같은 자체 기능이 없고 오직 버튼만 넣기 때문에 스크립트 함수 등을
 연결해 사용합니다.

 15) type="file" - 파일 첨부하기
