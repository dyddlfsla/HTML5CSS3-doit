
◆ 목록을 만드는 태그

목록을 만드는 태그는 실무에서 자주 사용되는 태그이므로 반드시 잘 익히고 넘어갑시다.

 1) <ul>, <li> - 순서 없는 목록 만들기
 순서가 필요하지 않은 목록(unordered list)을 만들 때는 <ul> 태그를 사용하고 <ul> 태그 안에
 <li> 태그를 사용해 각 항목을 표시합니다.
 순서 없는 목록의 경우, 각 항목 앞에 작은 원이나 사각형 같은 불릿(bullet)이 붙습니다. 이런 불릿은 css 의
 list-style-type 속성을 이용해 수정할 수 있습니다.

 2) <ol>, <li> 태그 - 순서 있는 목록
 항목을 나열하되, 순서가 필요한 목록(ordered list)를 만들 때는 <ul> 대신 <ol> 태그를 사용합니다.
 <ol> 태그를 사용할 때는 관련된 여러 속성들을 같이 사용할 수 있습니다.
 type 속성 → <ol> 태그는 기본적으로 순서 목록이기 때문에 각 항목 앞에 1,2 와 같은 숫자가 붙는데,
 이런 숫자를 알파벳이나 로마 숫자 등으로 바꿀 수 있습니다.
 start 속성 → 순서 목록은 기본적으로 1부터 시작하는데 start 속성을 이용하면 중간 번호부터 시작할 수 있습니다.
 reverse 속성 → 항목을 역순으로 표시합니다.
 * </li> 태그의 생략과 목록 중첩
 순서의 각 항목을 정의할 때 </li> 태그를 생략해도 <li> 태그가 다음에 오는 <li> 태그를 만나면 자동으로 </li>가 있는 것처럼
 인식합니다. 또한 목록의 항목들을 나열하는 도중에 또 다른 목록을 다시 나열할 수도 있습니다.

 3) <dl>, <dt>, <dd> - 설명 목록 만들기
 <dl> 태그는 사전 구성처럼 '제목'과 '설명'이 한 쌍인 설명 목록(description list)를 만듭니다.
 예를 들어 '단어/정의'나 '질문/답'처럼 사용할 수 있습니다. <dl> 태그는 목록을 만들고 <dt> 태그는 제목을, <dd> 태그는 설명을
 표시합니다.

 ※ 주석 - 소스코드에 설명 추가하기
 주석(comment) 란 나중에 자신이 작성한 HTML 소스를 보거나 다른 사람이 HTML 소스를 열어보더라도 해당 소스의 의미를 쉽게 이해할 수 있도록
 설명하는 글입니다. 주석은 사람을 위한 것이기 때문에 웹 브라우저에서 해석하지도 않고 나타나지도 않습니다. HTML 태그에서는 주석을 표시할 때
 <!-- --> 사이에 원하는 내용을 쓰면 되고 여러 줄에 걸친 주석도 가능합니다. css 에서 주석은 /* */ 사이에 입력합니다.

 ※ W3C HTML 검사기
 웹 브라우저에서 원하는 결과가 나왔더라도 HTML 문법에 맞게 작성되었는지, 웹 표준에 어긋나는 부분은 없는지 확인해보아야 합니다.
 웹 표준을 제정하는 W3C에서는 https://validator.w3.org 에서 온라인 HTML 검사기를 제공하고 있습니다.
