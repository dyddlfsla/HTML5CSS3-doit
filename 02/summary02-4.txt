
 ◆ 표를 만드는 태그
 웹 문서에서 자료를 정리할 때 자주 사용하는 요소가 '표(table)' 입니다. 표는 행(row) 와 열(column)으로 이루어져 있고
 행과 열이 만나 이루는 영역을 셀(cell)이라고 합니다. 표를 만드려면 행과 열 두 가지를 고려해야 하므로 하나의 표를
 만들기 위해서는 여러 개의 태그가 사용됩니다.

 1) <table>, <tr>, <td>, <th> - 기본적인 표 만들기
 기본적은 표의 모습을 상상해보세요. 가로 세로로 행과 열이 있고 그 안에 내용이 입력됩니다.
 이런 표를 만들 때는 표를 만드는 태그, 행을 만드는 태그, 열을 만드는 태그가 각각 필요합니다.

 <table>
   <tr>
    <td></td>
    <td></td>
   </tr>
   <tr>
    <td></td>
    <td></td>
   </tr>
 </table>

 ① 기본적인 표 만들기
 웹에서 표를 만들 때는 여러 태그가 사용되기 때문에 표 만들기를 연습하다 보면 태그들이 혼동되곤 합니다.
 웹 문서에서 표를 만들 때에는 순서가 있는데요. 이 순서를 기억해 두면 표 만들기가 한결 쉬워집니다.
 우선 <table> 태그를 이용해 표 전체 윤곽을 잡은 후 행(row)을 먼저 만들고 행 마다 몇 개의 셀이 들어갈지를 결정합니다.

 ② <th> 태그 - 표에 제목 셀 만들기
 표를 만들다 보면 첫번째 행이나 열에 셀의 제목을 넣는 경우가 많습니다. 제목의 경우, 보통 다른 글자보다 굵게 표시하고 셀의
 중앙에 배치하는데 <td> 태그 대신 <th> 태그를 사용하면 이런 과정을 간단히 줄이면서 제목 셀을 만들 수 있습니다.
 <th> 태그도 <td> 태그와 마찬가지로 셀을 만드는 태그로 해당 셀에 들어가느 내용을 셀의 중앙에 배치하고 굵게 표시합니다.

 ③ colspan, rowspan - 행 또는 열 합치기
 셀을 합치는 것은 셀을 만드는 <td> 태그나 <th> 태그에서 이루어집니다. 여러 열을 하나로 합치려면 <td> 태그나 <th> 태그 안에서
 colspan 속성을 사용해 몇 개의 셀을 합칠 지 지정합니다. 행을 합치는 것도 마찬가지입니다. <td> 태그나 <th>태그에서 rowspan 속성을
 이용해 세로로 합칠 셀의 개수를 지정하면 됩니다.

 2) <caption>, <figcaption> - 표에 제목 붙이기
 표에 제목을 붙일 때는 보통 <caption> 태그를 사용하지만 <figcaption> 태그를 사용할 수도 있습니다.

 ① <caption> 태그는 우선 <table> 태그 바로 다음에 사용합니다. 또한 <caption>태그를 사용한 표 제목은 표의 위쪽 중앙에 표시됩니다
 ② <figcaption> 태그는 <figure> 태그와 함께 쓰이는데, 설명 글을 붙이고 싶은 대상을 <figure> 태그로 우선 감싼 후 <figcaption> 태그를 이용해
 제목이나 설명 글을 입력합니다.

 ③ aria-describedby 속성 - 표에 대한 설명 제공하기
 화면 낭독기에서 표를 읽어줄 때 도움이 되도록 표 설명을 별도의 문장으로 작성한 후 <table> 안에 aria-describedby 속성을 추가해 연결하면
 표를 이해하는데 도움이 됩니다.

 3) <thead>, <tbody>, <tfoot> - 표 구조 정의하기
 일부 표에서는 제목이 표시된 셀과 자료가 표시된 셀 이외에도 표 아래 쪽에 합계나 요약 내용을 표시하기도 합니다. 이런 표의 각 셀은
 제목이 있는 부분과 실제 내용이 있는 본문 그리고 요약 부분이 있는 부분으로 표의 구조를 나누어 놓는 것이 좋습니다.

 <table>         이렇게 표의 구조를 정의하면 시각장애인들도 화면 낭독기를 통해 표의 구조를 쉽게 이해할 수 있습니다.
  <thead>
   <tr>
    <td></td>
  </tr>
  </thead>
  <tbody>
   <tr>
    <td></td>
   </tr>
  </tbody>
  <tfoot>
   <tr>
    <td></td>
   </tr>
  </tfoot>
 </table>


 4) <col>, <colgroup> - 여러 열 묶어 스타일 지정하기
